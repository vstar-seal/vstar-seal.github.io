Sense and Saliency

Machine learning models are so good at certain tasks they outperform humans. The canonical example is image classification: determining which object is depicted in an image. But how do these models correctly reocgnize objects?


Integrated gradients is a popular method to compute which features were most important towards the output prediction. We calculate the feature importance for every pixel in the input image \(x\) and plot the result below in a saliency map. Lighter pixels indicate more influential ones. As a baseline \(x'\), we use the constant black image. Some of the pixel attributions may seem unintuitive. But why? To understand why integrated gradients highlights the pixels it does, we will need to understand where the above equation comes from, and what it is computing.
